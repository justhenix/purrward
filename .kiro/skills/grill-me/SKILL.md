# Grill-Me

This skill triggers an interactive alignment interview to resolve design and architectural decisions.

## Workflow

1. Walk down each branch of the design tree to resolve dependencies between design, architecture, and feature decisions one-by-one.
2. Ask the questions **one at a time**.
3. For every question you ask, **provide your own recommended answer** first, explaining the rationale so the user can easily approve, reject, or customize it.
4. If a question can be answered by exploring the existing codebase, explore it first instead of asking.
5. Keep answers and questions extremely concise. Use simple options where possible.
