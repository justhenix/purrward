# Demo-only behavior

These flows are mocked or sandboxed for the hackathon build:

- Vet visit booking is mock demo flow, not a live appointment system.
- Sandbox proof accepts valid uploads without calling Gemini.
- Sandbox balance and proof points are intentionally high for local demos.

Do not present mock booking, demo coupons, or sandbox proof as production behavior.
