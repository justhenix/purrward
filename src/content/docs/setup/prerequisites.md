# Prerequisites

Install the basics first:

- Bun or Node.
- Git.
- A configured `.env` file.

Optional services:

- Google OAuth credentials for sign-in.
- Gemini API key for AI verification and AI Vet.
- Turso / LibSQL connection for persistent data.

Without optional credentials, some flows use sandbox behavior or show unavailable states.
